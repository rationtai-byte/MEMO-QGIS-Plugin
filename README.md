# MEMO Groundwater Network Design 💧

A professional QGIS plugin designed for environmental engineering and groundwater pollution site assessment. 

This tool evaluates the monitoring efficiency of groundwater well networks based on the **Domenico analytical transport model**, providing visual and quantifiable insights for site investigations.

## ✨ Core Features
* **Automated Efficiency Calculation:** Computes the interception efficiency of existing or proposed monitoring wells against potential pollution sources.
* **Customizable Hydrogeological Parameters:** Supports dynamic input of groundwater flow velocity, dispersivity (longitudinal & transverse), and source width.
* **Directional Flow Modeling:** Allows users to define groundwater flow direction, automatically adjusting the simulation and visualizing the flow with map arrows.
* **Blind Spot Visualization:** Generates semi-transparent polygons to highlight "Missed" areas (blind spots) and calculates the total area of monitoring gaps.
* **One-Click Export:** Exports all simulation results as a `.gpkg` (GeoPackage) with embedded symbology, alongside a bilingual (EN/zh-TW) summary report (`.txt`).

## 🚀 Use Cases
Designed for practical applications in soil and groundwater pollution investigations, such as optimizing well placements in industrial parks or generating scientifically backed efficiency reports for regulatory compliance.
